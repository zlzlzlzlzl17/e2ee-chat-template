# Limitations

- Experimental project with evolving behavior
- Android is the primary client in this template
- The legacy browser chat client is intentionally disabled in this template
- The active web surface is the management UI, not a general-purpose browser client
- Push notification behavior depends on Firebase and device/vendor background policies
- Voice calling is still experimental
- No multi-platform parity claim
- No security audit claim

Some Android vendors and regions may have unreliable background delivery even when the FCM setup is correct.
